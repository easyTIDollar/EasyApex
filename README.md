# EasyApex

EasyApex 是一款专为《Apex 英雄》玩家打造的轻量级战绩查询工具。


## 📥 下载安装

请前往 [Releases](https://github.com/easyTIDollar/EasyApex/releases) 页面下载最新的 APK 安装包。

## 🌟 功能亮点

- **智能查询**：支持通过玩家 Origin ID 或底层数字 UID 进行智能识别查询，解决部分老玩家搜不到 ID 的痛点。
- **实时数据**：查看玩家等级、实时在线状态（大厅/局内）、当前选用英雄及装备的追踪器数据。
- **游戏资讯**：实时同步官方地图轮换倒计时、各平台猎杀者入池底分及服务器运行状态。
- **隐私保护**：支持本地搜索历史记录，方便快速复查。


## 🛠️ 开发者说明

本项目使用 Kotlin + Jetpack Compose 构建，采用现代化的 MVVM 架构。

- **API 支持**：数据来源于 [apexlegendsstatus.com](https://apexlegendsstatus.com/) 非官方 API。
